## Installation & Usage

1. Clone the repository:
```bash
git clone https://github.com/yourusername/functional-diversity.git
```

2. Navigate to the project directory:
```bash
cd functional-diversity
```

3. Open `index.html` in your browser:
```bash
start index.html
```

Or simply open the file directly in your preferred web browser.
